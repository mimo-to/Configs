# Raycast

Raycast is a blazingly fast, extendable launcher that lets you complete tasks, calculate, share common links, and much more.

## Key Features:
- Script Commands
- Clipboard History
- Window Management
- System Commands
- Extensions Store

## Official Links:
- Website: https://www.raycast.com/
- Manual: https://manual.raycast.com/

## Installation:
- Download from official website
- Winget: `winget install Raycast` (Note: Windows version is currently in beta/development, verify availability)

## Configuration:
- Extensions and Store
- Hotkeys and Aliases
- Theme and Appearance
