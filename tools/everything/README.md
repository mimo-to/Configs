# Everything

Everything is a fast file search utility for Windows that indexes file names in real-time.

## Key Features:
- Lightning-fast file and folder search
- Real-time indexing
- Regular expression support
- Network drive search
- Export search results

## Official Links:
- Website: https://www.voidtools.com/
- GitHub (related tools): https://github.com/srwi/EverythingToolbar

## Installation:
- Download installer from official website
- Portable version available
- Settings stored in application directory

## Configuration:
- Index specific drives and folders
- Set file filters and search options
- Configure network search behavior
