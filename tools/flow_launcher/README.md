# Flow Launcher

Flow Launcher is a fast and customizable launcher for Windows with plugin support.

## Key Features:
- Application and file launcher
- Plugin ecosystem for extended functionality
- Web search integration
- Customizable themes and appearance
- Keyboard-centric operation

## Official Links:
- Website: https://www.flowlauncher.com/
- GitHub: https://github.com/Flow-Launcher/Flow.Launcher

## Installation:
- Microsoft Store: `winget install Flow-Launcher.Flow-Launcher`
- Chocolatey: `choco install flow-launcher`
- GitHub: Download from official repository

## Configuration:
- Settings stored in `%APPDATA%\FlowLauncher\`
- Custom hotkeys and themes
- Plugin configurations
