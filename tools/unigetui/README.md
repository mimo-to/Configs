# UniGetUI

UniGetUI is a modern package manager frontend for Windows with support for multiple managers.

## Key Features:
- Unified interface for WinGet, Chocolatey, Scoop
- Package search and management
- Update management
- Custom repository support

## Official Links:
- Website: https://www.marticliment.com/unigetui/
- GitHub: https://github.com/marticliment/UniGetUI

## Installation:
- Microsoft Store
- GitHub releases (MSIX format)

## Configuration:
- Package manager settings
- Repository configurations
- Theme preferences
