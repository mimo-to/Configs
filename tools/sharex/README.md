# ShareX

ShareX is a screen capture and file sharing tool for Windows.

## Key Features:
- Screenshot and video recording
- Image annotation tools
- Upload to cloud services
- Color picker and QR code generation
- File sharing and URL shortening

## Official Links:
- Website: https://getsharex.com/
- GitHub: https://github.com/ShareX/ShareX

## Installation:
- Download from official website
- Chocolatey: `choco install sharex`
- Scoop: `scoop install sharex`

## Configuration:
- Custom hotkeys for capture methods
- Upload destinations and services
- Image quality and naming conventions
