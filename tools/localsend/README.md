# LocalSend

LocalSend is a secure local file sharing tool that works across platforms.

## Key Features:
- Cross-platform file sharing
- End-to-end encryption
- No internet required
- No registration needed

## Official Links:
- Website: https://localsend.org/
- GitHub: https://github.com/localsend/localsend

## Installation:
- Windows: Available via package managers
- Android: Play Store or F-Droid
- iOS: App Store
- macOS/Linux: Available from GitHub

## Configuration:
- Portable mode support
- Network settings
- Security preferences
